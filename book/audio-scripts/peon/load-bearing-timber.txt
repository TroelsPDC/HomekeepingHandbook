Wood is strong. Big wood is more strong. This is how Morgum understand load-bearing. The load is there. The bearing is the wood's job. You find wood big enough for the job, you put it where the job is. This is the whole thing. Morgum will explain further but that is the whole thing.

When Morgum built the watchtower on south ridge — the second one, not the first one, the first one fall down — Morgum find big log from forest. Very big. Maybe this big [Morgum indicate width with both arms outstretched, which is not useful information in a written work, but picture very big]. Log not fit in hole Morgum dug. Morgum make hole bigger. Log fit in hole. Tower stand up for almost three season before burning down in unrelated incident. This is not the wood's fault. The wood did everything correct.

The important thing to remember is: if wood break, put more wood. This is not giving up. This is *learning*. The first wood told you something important — not enough wood. The second wood is you listening. Sometimes you must listen three, four times before wood is enough. This is fine. Every carpenter learn what enough wood is by watching what happen when there is not enough wood. Some carpenters learn this once. Morgum is not that kind. Morgum learn thorough.

If someone tell you that you must wait two years for the wood to be ready, ask them: ready for what? The wood was already in a tree before. The tree did not wait. The tree just *was* strong. The tree held the sky and the birds and all the ice of winter without anyone seasoning it first. You cut the tree down and now suddenly it need two years? The tree would hold up the ceiling right now. The tree would hold up the ceiling yesterday. This is what Morgum believe, and Morgum have built four watchtowers in support of this belief. Two of them are still standing.

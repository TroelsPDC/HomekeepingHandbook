# Audio Scripts

Narration scripts are available in two formats:

- Root files (`*.txt`): combined script per chapter (all voices)
- Voice folders: split scripts per chapter for easier upload

## Split by narrator

- `peasant/<chapter-slug>.txt` — Theodore sections only
- `peon/<chapter-slug>.txt` — Morgum sections only
- `acolyte/<chapter-slug>.txt` — Vorun sections only

Each voice folder contains one file per chapter slug.
